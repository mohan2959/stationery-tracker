# 📒 Stationery Business Tracker

A simple and modern web application designed to help small stationery shop owners track daily sales, monitor transactions, and analyze business performance.

---

## 🚀 Features

* 📊 **Sales Tracking** – Record daily transactions بسهولة
* 🧾 **Item Categories** – Organize sales (Books, Pens, Printing, etc.)
* 📅 **Date Management** – Track sales by day, week, and month
* 📈 **Visual Charts** – View spending and sales using charts
* 🔍 **Search & Filter** – Quickly find transactions
* 🧹 **Clear & Manage Data** – Delete or edit records easily
* 📤 **Export CSV** – Download your data for reports
* 💾 **Local Storage** – Data is saved in your browser automatically

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the app
* **CSS3 / TailwindCSS** – Styling and layout
* **JavaScript (Vanilla)** – Logic and interactivity
* **Chart.js** – Data visualization

---

## 📦 How It Works

1. Add a new sale using the **"Add Sale"** button
2. Fill in:

   * Date
   * Item name
   * Category
   * Price
   * Payment method
3. View:

   * Total sales
   * Monthly sales
   * Best-selling category
4. Analyze data using charts
